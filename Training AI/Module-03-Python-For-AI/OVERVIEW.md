# Module 3: Python for AI — Overview

## 5 Days of Lessons

| Day | Topic | What You'll Learn |
|-----|-------|-------------------|
| **Day 01** | NumPy Basics | Arrays, math on arrays, why NumPy is essential for AI |
| **Day 02** | NumPy Advanced | Reshaping, broadcasting, matrix operations |
| **Day 03** | Pandas Basics | DataFrames, reading CSV files, filtering data |
| **Day 04** | Pandas Advanced | Grouping, merging, cleaning messy data |
| **Day 05** | Matplotlib & Visualization | Plotting graphs, charts, visualizing AI data |

## Why This Module Matters
- **NumPy** = ALL AI math runs on NumPy arrays (not regular Python lists)
- **Pandas** = Every dataset you'll work with uses Pandas
- **Matplotlib** = You MUST visualize data before training AI

These 3 libraries are used in EVERY AI project. Master them.
