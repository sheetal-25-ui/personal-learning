# Module 6: Deep Learning — Overview

## Welcome to Neural Networks!

## 6 Days of Lessons

| Day | Topic | What You'll Learn |
|-----|-------|-------------------|
| **Day 01** | What is a Neural Network? | Neurons, layers, activation functions — explained simply |
| **Day 02** | Building Your First Neural Network | Hands-on with PyTorch — classify handwritten digits |
| **Day 03** | Training Deep Networks | Loss functions, optimizers, backpropagation (intuitive!) |
| **Day 04** | CNNs (Convolutional Neural Networks) | How AI "sees" images — image classification project |
| **Day 05** | RNNs and Sequence Models | How AI understands sequences — text, time series |
| **Day 06** | Practical Deep Learning | Transfer learning, data augmentation, tips and tricks |

## Tools You'll Use
- **PyTorch** — The AI framework used by Anthropic, Meta, Tesla
- **Google Colab** — Free GPU for training

## Why PyTorch (Not TensorFlow)?
- Anthropic uses PyTorch
- More intuitive and Pythonic
- Preferred for research and cutting-edge work
- Most new AI papers use PyTorch
