# Module 5: Machine Learning Basics — Overview

## This is the CORE of your AI journey!

## 7 Days of Lessons

| Day | Topic | What You'll Learn |
|-----|-------|-------------------|
| **Day 01** | ML Workflow | The complete ML pipeline: data → train → evaluate → deploy |
| **Day 02** | Linear Regression | Predict numbers (house prices, salary) — your first ML model! |
| **Day 03** | Logistic Regression | Predict yes/no (spam detection, disease prediction) |
| **Day 04** | Decision Trees & Random Forest | Visual models that make decisions like a human |
| **Day 05** | Model Evaluation | Accuracy, precision, recall, F1 — how to know if your model is good |
| **Day 06** | Feature Engineering | Making your data better so your model learns better |
| **Day 07** | Full ML Project | End-to-end project: predict something real with real data |

## Tools You'll Use
- **scikit-learn** — The most popular ML library
- **Google Colab** — Free cloud environment with GPU
- **Kaggle** — Real datasets for practice

After this module, you can build real ML models and put them on your resume!
