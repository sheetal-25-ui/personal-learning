## How This Course Works

- **17 Modules** in 3 phases — each builds on the previous one
  - **Phase 1 (Modules 1-4):** Foundations — think + code
  - **Phase 2 (Modules 5-10):** Core AI — build real AI + get job-ready
  - **Phase 3 (Modules 11-17):** Mastery — DSA, NLP, Vision, RL, Generative AI, MLOps, System Design
- **Each module** has lessons (Day 1, Day 2, etc.)
- **Each lesson** takes about **1-2 hours**
- **Every lesson** has: Explanation + Examples + Practice Exercises
- **Ask me anytime** — I'm your teacher, ask Claude to explain anything!
