| **5 min** | Check solution if stuck, understand the approach |

| **Next 25 min** | Solve Question 3 |
| **Last 15 min** | Revise — write the pattern in your own words |

---

## Rules

1. **Try every question for 20-25 minutes** before looking at solution
2. **If you can't solve it** — read the approach (not full code), then try again
3. **After solving** — ask yourself: can I make it faster? less memory?
4. **Revise weekly** — redo 5 questions from previous weeks every Sunday
5. **Write patterns, not solutions** — understand WHY, not just HOW
6. **Use Python** — same language as your AI prep

## How to Use LeetCode

1. Go to leetcode.com → create free account
2. For each question below, the link takes you directly to it
3. Select **Python 3** as your language
4. Try to solve → Run → Submit
5. After solving, read the "Discuss" tab for better approaches
