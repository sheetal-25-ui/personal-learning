# Week 10: Mock Interviews

## Simulate Real Microsoft Interviews

### Daily Mock Format (2 hours)

```
First 45 min:  Solve 2 questions (like a real interview round)
Next 15 min:   Review — did you explain clearly? Any bugs?
Next 45 min:   Solve 2 more questions
Last 15 min:   Review + note weak areas
```

## You're Ready When...

- [ ] Can solve 2 Medium problems in 45 minutes
- [ ] Can explain your approach clearly before coding
- [ ] Rarely get stuck for more than 5 minutes
- [ ] Know at least 2 approaches for common problem types
- [ ] Can handle "what if..." follow-up questions
- [ ] Feel confident, not anxious, when seeing a new problem
