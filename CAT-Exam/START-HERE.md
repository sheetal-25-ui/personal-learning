# CAT Exam Preparation — Complete Course

## CAT Exam Overview

| Detail | Info |
|--------|------|
| **Full Form** | Common Admission Test |
| **Purpose** | Admission to IIMs and top MBA colleges |
| **Conducted By** | IIMs (rotates each year) |
| **Exam Duration** | 2 hours (120 minutes) |
| **Total Questions** | 66 questions |
| **Total Marks** | 198 marks (3 marks each) |
| **Negative Marking** | -1 for wrong MCQ answer, no negative for TITA (Type In The Answer) |
| **Mode** | Online (Computer-Based Test) |
| **Sections** | 3 (VARC, DILR, Quant) |
| **Time per Section** | 40 minutes each (can't switch between sections) |

## Golden Rules for CAT

1. **Mocks > Theory** — After basics, solving mocks teaches more than reading theory
2. **Mock Analysis > Mock Score** — Reviewing WHY you got wrong is 10x more important than the score
3. **Accuracy > Attempts** — Attempting 40 with 85% accuracy beats attempting 55 with 60% accuracy
4. **Time Management** — Skip hard questions, come back later. Don't spend 8 min on 1 question.
5. **Sectional Cutoffs** — You must clear cutoff in ALL 3 sections, not just overall
